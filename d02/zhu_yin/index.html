    # 通用拼音 (opens new window)

    通用拼音是源自臺灣的拉丁字母漢字標音方案，包含了「華語通用拼音」、「臺語通用拼音」、「客語通用拼音」三種拼音方案。一般情況下，通用拼音常用做「華語通用拼音」的簡稱，也是中華民國政府在2002年至2008年採用的中文拉丁化拼音法。中華民國教育部於2008年12月18日修正「中文譯音使用原則」，改採漢語拼音為標準。目前，部分縣市政府使用通用拼音翻譯地名。
