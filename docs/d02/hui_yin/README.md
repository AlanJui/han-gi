### 拚音

__【反切法拚音規則】__

如何使用《四十五字母音》（韻）、《八聲調》及《十五音》（聲），進行「反切」，拚出「字」的讀音。

 - 緣：【堅五英／ian5】
 - 平：【經五邊／ping5】
 - 安：【干一英／an1】
